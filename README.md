# Huzz-client
A simple add-on that aims to improve quality of life and ease finding bases on Minecraft servers. Mainly aimed at DonutSMP.

Current modules as of version 3.0.0 - 

Render - 
  *Chunk finder(possibly not working)
  *Time debug(possibly not working)
  *Prime chunk(possibly not working)
  *Block ESP
  *Storage ESP
  *Player ESP
  *Full bright
  *Item tags
  *Hud

Movement - 
  *Free cam
  *Free look
  *Fast use
  *Auto totem(Auto inventory totem)

Player - 
  *Player notify
  *Fake media
  *Fake pay
  *Fake stats
  *Name protect

Utility - 
  *Stash finder
  *Block notify
  *Crafter macro - auto selects sqaures inside of a crafter

Modules planned for next update - 
  
  *Skin protect
  *Swing speed
  *Chat macro
  *Auto rocket
  *Hole esp
  *Anti chunk finder - finds empty chunks to build bases in

Patches planned for next update -
