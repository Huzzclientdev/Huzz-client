# Huzz-client 1.21.11
A simple add-on that aims to improve quality of life and ease finding bases on Minecraft servers. Mainly aimed at DonutSMP.

Current modules as of version 3.3.1 - 

Render - 
  *Block ESP
  *Storage ESP
  *Hole ESP
  *Player ESP
  *Full bright
  *Item tags
  *Hud
  *Break bar
  *Motion blur
  *Block select
  *Custom fov
  *Fake elytra
  *Fake spawner - work in progress

Movement - 
  *Swing speed
  *Free cam
  *Free look
  *Fast use
  *Auto totem(Auto inventory totem)
  *Auto rocket
  *Auto refill
  *Auto bridge
  *Double click
  *Auto crystal
  *Auto tool

Player - 
  *Player notify
  *Auto disconnect
  *Skin protect
  *Fake media
  *Fake pay
  *Fake stats
  *Name protect
  *67

Utility - 
  *Auto reconnect
  *Stash finder
  *Block notify
  *Chat macro
  *Crafter macro - auto selects sqaures inside of a crafter
  *Silent sethome - work in progress
